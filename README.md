## About .ipynb Files

All code in this repository is **written in C**. The `.ipynb` (Jupyter Notebook) format is used here as a reference and documentation tool. Each notebook contains:
- **Problem Statement** - The complete LeetCode question in markdown
- **Solution Code** - C language implementation in a code cell

This format allows for easy viewing, reference, and version control of both problem descriptions and solutions in a single file.

## Problem Categories

### Linked List
- **Easy Problems**: 12 problems covering fundamentals like reversal, cycle detection, merging, and traversal
- **Medium Problems**: 14 problems covering advanced operations like node deletion, sorting, and complex transformations
